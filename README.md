# Buckshot Roulette Style Python Game

A buckshot inspired game made with **Python**

## Gameplay

Classical Russian roulette but with **items.**

## Features

- **10** different items to use.
- The game holds a few **secrets...** Try experimenting with different actions to uncover hidden easter eggs. **Can you find them all?**

## Install Required Libraries
```
python -m pip install -r requirements.txt
```

## How to Play?

**After installing the required libarys,** just clone this repository and run buckshot.py
